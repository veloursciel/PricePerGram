# プロジェクト基本情報

このプロジェクトは HTML/JavaScript/CSS で書かれた Web アプリケーションです。
「1gあたりの単価」を比較しどの商品が最安かを計算します。

## プロジェクト内容

- 表形式でユーザーが'重さ'，’価格’を入力し、'1gあたり'を自動計算する．

### 表について

- 3*3の表を用意する．
- 表は内枠のみ罫線表示する．外枠には何も表示しない．
- 1行目は'重さ'，'価格'，'1gあたり'のラベルを表示する．
- 2行目以降の’重さ’，’価格’列にユーザーが自然数を入力する．
- 2行目の'価格'列で改行をすると3行目の’重さ’セルに移動する．
- 3行目の'価格'列で改行をすると行を追加する機能を実装する．
- ’価格’列の入力を終了する際に'1gあたり'を自動計算する．
- 最安の商品は’1gあたり’を赤字で表示する．
- かっこいいみためのカラーは青系統で統一


## 実装指示

- フォント等のスタイルについてはCSSファイル内に全て実装し，HTMLファイル内には書き込まないこと．
